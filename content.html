Wasteland Survival Guide

Content

First, create a "Content" separator in MO2.

As you install the mods from this section, organize them underneath this separator.
Quests, Locations & NPCs
A World of (Less) Pain
A Lore Friendly AWOP Revision 		An edit of the original A World of Pain that removes all unvoiced NPCs, quests, un-lore friendly interiors, and "M" weapons, ending up with about 60 rebalanced locations Main File - AWOP Revised 3.4
FPGEL
Functional Post Game Ending Lite 		A stripped down version of FPGE that doesn't alter the game's world after the Second Battle of Hoover Dam. Main File - Functional Post Game Ending Lite TTW 2.6
Metro Carriage Interiors TTW Allows the player to enter wrecked metro cars within DC Metro stations and above ground train yards. It also grants a means of reaching and entering several monorail cars around the DC wasteland. Main File - Metro Carriage Interiors TTW 1.1
Rebuild the Capital
A Brotherhood of Steel Expansion Mod
  • Expand the Brotherhood of Steel and their role in the wasteland following the events of Broken Steel.
  • Under the player's guidance, the Brotherhood can establish relations with settlements and establish military outposts.
  • The mod also features new armors, weapons, quests, perks, NPCs, and a recruitment system.
  • Main File - Rebuild the Capital 2.02
  • Update File - Rebuild the Capital 2.04
When installing the update you can just click
Sweet Home Explore an old rivalry between a notorious terrorist of the Wastes and an old ghoul that seeks vengeance. Make a fateful decision which could shape the Wastes forever. What will you decide? Who will live? Who will you believe? Main File - Sweet Home 12
The Living Desert
Travelers Patrols Consequences
Increased Population and More
  • Adds hundreds of NPCs and several scripted events to the world of Fallout: New Vegas
  • NPCs patrol roads, travel between towns, occupy locations, explore the desert
  • They will react dynamically to your choices throughout the game
    		•  Main File - The Living Desert 2.3
    Trooper's Truce Series A series of plot/story mods which allow you to form allegiances between previously incompatible groups such as the BoS & Legion or House & NCR
    All of Trooper's other mods outside this series are also compatible & recommended - check his profile.
    		 Options include:
    • Brotherhood & House Truce
    • Brotherhood & Legion Truce
    • Hardin and NCR Truce
    • House & Followers Truce
    • House & Kings Truce
    • House & NCR Truce
    It is impractical and unnecessary to use all of these mods simultaneously. Only install the ones you actually intend to use on this playthrough
    TTW A Trail of Crumbs An exciting, fun and challenging quest to 10 forgotten locations Main File - A Trail Of Crumbs 1.5
    TTW Another Interior Mod Adds 40+ locations to the DC Wasteland;one compatible with AWOP4FO3 & The Regulators. Main File - Another Interior Mod 1.0
    TTW Bobblehead Quests Adds two quests, once you leave Vault 101 (or reach DC from a Mojave start), that add quest markers for the SPECIAL and Skill Bobbleheads Main File - TTW Bobblehead Quests 1.1
    TTW Steel Ingot Quest Adds a quest that starts when Midea tells you to go collect Steel Ingots that adds quest markers for all 100 Steel Ingots Main File - TTW Steel Ingot Quest 1.0
    TTW The Regulators Having joined the Regulators after getting the Lawbringer perk, you notice a computer in the downstairs room. From it, you accept contracts on fugitives and their gangs all over the wasteland. Its time for some wasteland justice! Main File - The Regulators 1.5
    Vikki and Vance Improvements Improve some aspects of the Vikki & Vance casino so it actually looks like a casino while staying true to the vanilla aesthetic Main File - Vikki and Vance Improvements 1.4
    Uncut Wasteland
  • Restores a huge amount of scenery and other details which were patched out New Vegas post-release
  • Fluff NPCs and creatures are restored too
    		•
    • Main File - Uncut Wasteland plus NPCs. 0.91b
    • Optional File - Uncut Extra Collection 0.92.2
    • Patch - ttw - uncut wasteland plus npcs patch 1.0
    • Patch - uncut extra collection - ttw patch 1.0
    Washington's Malevolence TTW A huge quest mod where the Lone Wanderer travels to the fictional Maddox Island in search of the Rocky Cape Bank, a pre-War bank that is said to contain untold riches. Players will also get to explore multiple locations around the island with interesting backstories and meet plenty of different personalities that are sure to leave an impression that lasts. Main File - Washington's Malevolence TTW 3.2.3
    Player Housing
    A Very Uncreative Jacobstown Bungalow Adds a basic small player home in Jacobstown. It has some containers, crafting stations, a bed, and water sources Main File - A Very Uncreative Jacobstown Bungalow 5.0
    Dynamic Weapon Displays
    • Adds Skyrim-esque weapon displays
    • Fully compatible with weapons from all the DLC and weapons added by other mods
    • Contains patches for TTW and modded player homes including the Jacobstown Bungalow
    • Main File - Dynamic Weapon Displays 1.2
    • In the FOMOD installer select:
      • Vanilla Player Homes - All In One
      • TTW Player Homes - All In One
      • Modded Player Homes - A Very Uncreative Jacobstown Bungalow
      • Weapon Display Models - Default
    Home & Safehouse Tweaks Adds crafting stations and additional containers to several vanilla player homes (safehouses, suites, rooms, etc) Main File - Home and Safehouse Tweaks 1.4
    Mobile Truck Base
    TTW Community Preview Edition     		Bomb around the wastes in this sweet mobile player home that consumes flamer fuel. TTW edition; can travel between wastelands Click the mod name to download the main file and install in MO2.
    The truck is located in Scrapyard (DC) and the key is nearby in a toolbox
    Own The Ultra-Luxe Penthouse A mod that allows you to gain ownership after completing Beyond the Beef Main File - Own The Ultra-Luxe Penthouse 6.1
    TTW Red Rocket Workshop Player Home A small player home located in the basement of a new building in Springvale, at the Red Rocket gas station Main File - TTW Red Rocket Workshop Player Home 1.2
