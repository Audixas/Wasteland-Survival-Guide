Wasteland Survival Guide

Content

First, create a "Content" separator in MO2.

As you install the mods from this section, organize them underneath this separator.

Rebuild the Capital
A Brotherhood of Steel Expansion Mod
  • Expand the Brotherhood of Steel and their role in the wasteland following the events of Broken Steel
  • Under the player's guidance, the Brotherhood can establish relations with settlements and establish military outposts
  • The mod also features new armors, weapons, quests, perks, NPCs, and a recruitment system
  • Main File - Rebuild the Capital 2.02
  • Update File - Rebuild the Capital Hotfix 2.08
    • In the FOMOD check No Pony Express Boxes (or don't; it is a personal preference), then click

Sweet Home Explore an old rivalry between a notorious terrorist of the Wastes and an old ghoul that seeks vengeance. Make a fateful decision which could shape the Wastes forever. What will you decide? Who will live? Who will you believe? Main File - Sweet Home 12
The Living Desert
Travelers Patrols Consequences
Increased Population and More
  • Adds hundreds of NPCs and several scripted events to the world of Fallout: New Vegas
  • NPCs patrol roads, travel between towns, occupy locations, explore the desert
  • They will react dynamically to your choices throughout the game
  • Main File - The Living Desert 2.3
  • AWOLP/Living Desert Compatibility Patch - Underpass Patch - All Mods 1.0
New Vegas Truce Series This is a series of mods that allows you to ally factions whom were previously incompatible. This is a compilation of my previously released Truce mods. All the mods have been compiled into a single, easy to install FOMOD
  • Main File - New Vegas Truce Series 1.0
  • You will be prompted with a FOMOD installer in which you should ideally only check the ones you actually plan to use this playthrough
TTW Merged Collectable Quests Adds quest markers for collectibles including: bobbleheads, steel ingots, alien captive recordings, intel briefcases & Keller family transcripts Main File - TTW Merged Collectable Quests 1.0
