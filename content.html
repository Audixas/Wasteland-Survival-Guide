Wasteland Survival Guide

Content

First, create a "Content" separator in MO2.

As you install the mods from this section, organize them underneath this separator.
Mod Name Description Instructions
The Living Desert
Travelers Patrols Consequences
Increased Population and More
  • Adds hundreds of NPCs and several scripted events to the world of Fallout: New Vegas
  • NPCs patrol roads, travel between towns, occupy locations, explore the desert
  • They will react dynamically to your choices throughout the game
    		•  The Living Desert - Main File 2.3
    Uncut Wasteland
  • Restores a huge amount of scenery and other details which were patched out New Vegas post-release
  • Fluff NPCs and creatures are restored too
    		•
    • Main File - Uncut Wasteland plus NPCs. 0.91b
    • Optional File - Uncut Extra Collection 0.92.2
    • Patch - ttw - uncut wasteland plus npcs patch 1.0
    • Patch - uncut extra collection - ttw patch 1.0
    Abandoned Safehouse TTW
    • Adds a player home similar in layout to the NCR Ranger Safehouse to Springvale
    • Located behind the Red Rocket Pitstop by the dumpsters, locked with a very easy lock
    		Main File - Abandoned Safehouse TTW 3.0a
    AVeryUncreativeTrooper's Truce Series A series of plot/story mods which allow you to form allegiances between previously incompatible groups such as the BoS & Legion or House & NCR Options include:
    • Brotherhood & House Truce
    • Brotherhood & Legion Truce
    • Hardin and NCR Truce
    • House & Followers Truce
    • House & Kings Truce
    • House & NCR Truce
    • Three Familes Alliances
    It is impractical and unnecessary to use all of Trooper's mods simultaneously. Only install the ones you actually intend to use on this playthrough.
    A Very Uncreative Jacobstown Bungalow Adds a basic small player home in Jacobstown. It has some containers, crafting stations, a bed, and water sources. Main File - A Very Uncreative Jacobstown Bungalow 5.0
    Dynamic Weapon Displays
    • Adds Skyrim-esque weapon displays
    • Fully compatible with weapons from all the DLC and weapons added by other mods
    • Contains patches for TTW and modded player homes including the Jacobstown Bungalow
    • Main File - Dynamic Weapon Displays 1.2
    • In the FOMOD installer select:
      • Vanilla Player Homes - All In One
      • TTW Player Homes - All In One
      • Modded Player Homes - A Very Uncreative Jacobstown Bungalow
      • Weapon Display Models - Default
    Home & Safehouse Tweaks Adds crafting stations and additional containers to several vanilla player homes (safehouses, suits, rooms, etc) Home and Safehouse Tweaks 1.4
    Own The Ultra-Luxe Penthouse A mod that allows you to gain ownership after completing Beyond the Beef. Main File - Own The Ultra-Luxe Penthouse 6.1
    TTW Red Rocket Workshop Player Home
  • A small player home located in the basement of a new building in Springvale, at the Red Rocket gas station.
    		• Main File - TTW Red Rocket Workshop Player Home 1.2
