Wasteland Survival Guide

User Interface & HUD

First, create a "UI & HUD" separator in MO2.


As you install the mods from this section, organize them underneath this separator.

Mod Name Description Instructions
The Mod Configuration Menu
(a.k.a. MCM)
  • Settings menu to configure mods in-game
  • Required for many mods
 Normal MO2 installation.
Also install the Optional File* MCM BugFix 2 normally through MO2, clicking when you do.
* It's not optional, don't let that fool you.
Vanilla UI Plus
(a.k.a. VUI+)
  • Modernized vanilla-style UI
  • Many enhancements and bugfixes
 Normal MO2 installation.
Choose this UI if you prefer to keep the original style of the UI, only without its numerous shortcomings, bugs and limitations.
Download Vanilla UI Plus - Custom Settings and install as a new mod in MO2, placing it under the Config separator. This will enable:
  • 9 dialogue topics displayed at a time (from 5)
  • Numbered dialogue topics for use with Tweaks hotkeys 1-9
  • Darkened dialogue background
  • 11 item rows (from 10)
  • Dynamic condition bar
  • Minimalistic activation prompt
Feel free to customize in a text editor like Notepad++.
Darnified UI (a.k.a. DarnUI)
(or just Darn)
  • Alternative, smaller UI with retrofuturist style
  • Customizable extra HUD info (H20, RADS, etc.)
Choose this UI if you want something different, or find VUI+ too big. This is an alternative to VUI+; don't try to use both at the same time!
 Manual download and install.
Be sure to read the thread carefully! Darnified UI requires special font tweaks to install correctly.
Vanilla HUD Cleaned High-quality HUD (use with VUI+). Normal MO2 installation. Not compatible with Darn.
Consistent Pip-Boy Icons High-quality icons that are used for perks, items, etc. Normal MO2 installation.
Grab any of the optional files you like as well, and them into the main mod.
JIP Minimap Customizable realtime minimap that appears on the HUD. Normal MO2 installation. Configure in MCM.
This mod has some minor issues with certain mods such as Unfound Loot. It's nothing gamebreaking, but do be advised.
AidUI
(Active Ingestibles Duration UI)		 Adds widgets on HUD for consumed Aid items that display the time until their effects run out. Normal MO2 installation. Configure in MCM - bars VS icons, enable or disable food/alcohol/chems, etc.
