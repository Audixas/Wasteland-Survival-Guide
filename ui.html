Wasteland Survival Guide

User Interface & HUD

First, create a "UI & HUD" separator in MO2.

As you install the mods from this section, organize them underneath this separator.

Mod Name Description Instructions
User Interface Organizer
  • Automatically ensures that multiple HUD extensions work together
  • Required to use the following UI mods
 Main File - User Interface Organizer
The Mod Configuration Menu
  • Settings menu to configure mods in-game
  • Required for many mods
  • Main File - The Mod Configuration Menu
  • Update File - MCM BugFix 2
Do not neglect to install the update file! The MCM will not function without it.
Pick only one of the following two UI mods, VUI+ or DarN. These will determine your primary HUD and user interface so compare both and make a decision
Vanilla UI Plus Modernized vanilla-style UI with many enhancements and bugfixes
Choose this UI if you prefer to keep the original style of the UI, only without its numerous shortcomings, bugs and limitations
  • Click to download then install normally in MO2.
    If you don't know how to install a mod that isn't from Nexus, refer to the Mod Installation Advice at the end of the MO2 section of the guide
  • You will be prompted with a FOMOD installer
  • Tick the Plugin & WASD Compatible options & click
  • Guide File - VUI Plus Fonfig
Darnified UI TTW Alternative, smaller UI with retrofuturist style featuring customizable extra HUD info (H2O, RADS, etc.)
Choose this UI if you want something different, or find VUI+ too large
  • Download the Main File from MEGA link; install normally in MO2
  • Do the same with Optional File - MCM Matches HUD
AidUI - Active Ingestibles Duration Adds widgets on HUD for consumed Aid items that display the time until their effects run out
  • Scroll down to the Download link; install AidUI
  • In the MCM in-game, configure the settings to match this image
    • Your positioning may be different depending on your resolution, adjust as needed so the icons don't overlap with other UI elements such as JIP Minimap
Character Preset Menu Integrates character import and export into the character creation menu. Share your designs with others! Main File - Character Preset Menu
Clean Vanilla HUD Hand-made up-scaled/cleaned recreations of many vanilla HUD elements
If you use DarN UI, take extra care that you do NOT accidentally install the Vanilla UI+ Patch for this mod - this combination will lead to a crash
  • Main File - Vanilla Hud Cleaned
  • You will be prompted with a FOMOD installer:
    • If you use VUI+, check all boxes EXCEPT:
      • Clean Fonts for DarnUI
    • If you use DarnUI, check all boxes EXCEPT:
      • Clean Fonts
      • Clean Radiation Bar
      • Vanilla UI+ Patch
  • Clean Companion Wheel - Main File - Clean Companion Wheel
Consistent Pip-Boy Icons High-quality icons that are used for perks, items, etc.
You may be wondering about the third main file, CPI - Mod Patches. We are in fact going to install it, but later on in the Aesthetics section of the guide. The VUI+ patch for this mod isn't needed either, even if you use VUI+. A functionally identical patch comes with the previous mod, Clean Vanilla HUD
  • Main File - 1. Consistent Pip-boy Icons
  • Main File - 2. Consistent Addon Icons
    • In the FOMOD just click
  • Pop-Up Message Icons - Main File
JIP Minimap Adds a customizable mini-map to the HUD display with quest targets, map & door markers, etc
  • Main File - JIP MiniMap
  • Guide File - JIP Minimap Fonfig
  • Press N to toggle the map on & off; press-and-hold it to switch between local and world map modes
Main Menu Redone - TTW A completely reworked re-imagining of the TTW main menu image, beginning with higher resolution base photos, AI upscaling to 4k, and then custom photo retouching and masking, resulting in a much higher quality version Main File - Main Menu Redone - TTW - 1080p
Obviously, you should use the 1440p or 2160p instead if one of those matches your game resolution
MAPMO - Main & Pause Menus Overhaul Adds a bunch of features to the Main and Pause menus. Fully customizable
With this mod installed, the Mod Configuration Menu button in the pause menu will be hidden! You will have to access the MCM by pressing the M key. Please do your best to keep this in mind & do not report this as a bug
  • Main File - Main And Pause Menus Overhaul
  • Guide File - MAPMO Preset - WSG
  • Press M from the pause menu to access the MCM
  • Press N from the pause menu to configure this mod's settings
Skill Book Checklist Adds an MCM checklist of the skill books in both wastelands & can optionally notify you when you enter a cell containing one Main File - Skill Book Checklist
Vault Door Loading Wheel Simple, non-Casino-themed loading wheel replacer aimed at TTW users Main File - Vault Door Loading Wheel - TTW
yUI - User Ynterface NVSE plugin that aims to fix UI bugs and add new UI features
  • Main File - yUI
    • VUI+ users: Optional File - Matched Cursor - Fallout New Vegas
    • DarN users: Optional File - Matched Cursor - DarN UI
  • Guide File - yUI Fonfig
ySI - Sorting Ycons
  • Sorting mod using features of yUI to make inventory management a better experience
  • Each item gets assigned one of the 140+ included custom icons, representing different categories of items
 Main File - ySI - Sorting Icons
Requires bSortingIcons = 1 in yUI.ini.
It has been enabled for you in the yUI Fonfig provided above
