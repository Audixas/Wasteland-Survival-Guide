Wasteland Survival Guide

User Interface & HUD

First, create a "UI & HUD" separator in MO2.

As you install the mods from this section, organize them underneath this separator.

Mod Name Description Instructions
UIO
User Interface Organizer
  • Automatically ensures that multiple HUD extensions work together
  • Required to use the following UI mods
 Main File - User Interface Organizer 2.30
MCM
The Mod Configuration Menu
  • Settings menu to configure mods in-game
  • Required for many mods
  • Main File - The Mod Configuration Menu 1.5
  • Update File - MCM BugFix 2
Do not neglect to install the update file! The MCM will not function without it.
VUI+
Vanilla UI Plus
  • Modernized vanilla-style UI
  • Many enhancements and bugfixes
Choose this UI if you prefer to keep the original style of the UI, only without its numerous shortcomings, bugs and limitations.
  • Click to download then install normally in MO2.
  • You will be prompted with a FOMOD installer
  • Tick the Plugin & WASD Compatible options & click
DarnUI
Darnified UI TTW
  • Alternative, smaller UI with retrofuturist style
  • Customizable extra HUD info (H20, RADS, etc.)
Choose this UI if you want something different, or find VUI+ too large. This is an alternative to VUI+; don't try to use both at the same time!
  • Download the Main File from the provided mega link
  • Download the Optional File - MCM Matches HUD as well.
Get the Optional file MCM matches HUD too.
VHC
Vanilla Hud Cleaned		 Hand-made up-scaled/cleaned recreations of many vanilla HUD elements
  • Main File - Vanilla Hud Cleaned
  • You will be prompted with a FOMOD installer
    • If you use VUI+, select every option EXCEPT Clean Fonts for Darn
    • If you use DarnUI, select every option including Clean Fonts for Darn.
  • Download the Update File - Vanilla Hud Cleaned - Barter Update and install normally through MO2
CPI
Consistent Pip-Boy Icons		 High-quality icons that are used for perks, items, etc. Normal MO2 installation.
Grab any of the optional files you like as well, and them into the main mod.
AidUI
Active Ingestibles Duration		 Adds widgets on HUD for consumed Aid items that display the time until their effects run out.
  • Scroll down to the Download link; install AidUI 2.0
  • Configure the mod to your liking in the MCM.
    • I prefer the Bars mode over the Icons personally.
JIP Minimap Customizable realtime minimap that appears on the HUD. Normal MO2 installation. Configure in MCM.
The HUD Editor
  • Allows you to rearrange, resize and otherwise edit the HUD.
  • Watch the video in the description for a usage demonstration.
    		•  Main File - The HUD Editor 2.5.4
    If you are a DarnUI user, you need the
    Optional File - Darnified UI - TTW (Patch-v1)
    yUI
    User Ynterface     		NVSE plugin that aims to fix UI bugs and add new UI features
    • Main File - yUI 1.4c
    • Download this W.S.G. File - yUI INI and install after the main file.
    ySI
    Sorting Ycons
    • Sorting mod using features of yUI to make inventory management a better experience.
    • Each item gets assigned one of the 140+ included custom icons, representing different categories of items.
    		 Main File - ySI - Sorting Icons 1.3
    Requires bSortingIcons = 1 in yUI.ini - it has been enabled for you in the provided INI above.
