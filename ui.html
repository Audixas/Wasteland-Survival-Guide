Wasteland Survival Guide

User Interface & HUD

First, create a "UI & HUD" separator in MO2.

As you install the mods from this section, organize them underneath this separator.

Mod Name Description Instructions
User Interface Organizer
  • Automatically ensures that multiple HUD extensions work together
  • Required to use the following UI mods
 Main File - User Interface Organizer 2.30
The Mod Configuration Menu
  • Settings menu to configure mods in-game
  • Required for many mods
  • Main File - The Mod Configuration Menu 1.5
  • Update File - MCM BugFix 2
Do not neglect to install the update file! The MCM will not function without it.
Pick one of the following two UI mods, not both! These will determine your primary HUD and user interface so compare both and make a decision.
Vanilla UI Plus
  • Modernized vanilla-style UI
  • Many enhancements and bugfixes
Choose this UI if you prefer to keep the original style of the UI, only without its numerous shortcomings, bugs and limitations.
  • Click to download then install normally in MO2.
  • You will be prompted with a FOMOD installer
  • Tick the Plugin & WASD Compatible options & click
  • Guide File - VUI Plus Fonfig
Darnified UI TTW
  • Alternative, smaller UI with retrofuturist style
  • Customizable extra HUD info (H2O, RADS, etc.)
Choose this UI if you want something different, or find VUI+ too large. This is an alternative to VUI+; don't try to use both at the same time!
  • Download the Main File from the provided mega link
  • Download the Optional File - MCM Matches HUD as well.
AidUI - Active Ingestibles Duration Adds widgets on HUD for consumed Aid items that display the time until their effects run out
  • Scroll down to the Download link; install AidUI 2.0
  • In the MCM in-game, configure the settings to match this image
    • Your positioning may be different depending on your resolution, adjust as needed so the icons don't overlap with other UI elements such as JIP Minimap
Consistent Pip-Boy Icons High-quality icons that are used for perks, items, etc.
  • Main File - 1. Consistent Pip-boy Icons
  • Main File - 2. Consistent Add-On Icons
    • In the FOMOD just click
  • Main File - 3. Consistent Glow Icons
  • Optional File - 5. CPI - Alternative Icons
    • You may be wondering about the 4th main file - we are in fact going to install it, but later on in the guide
Character Preset Menu Integrates character import and export into the character creation menu. Share your designs with others! Main File - Character Preset Menu 1.0
JIP Minimap Adds a customizable mini-map to the HUD display with quest targets, map & door markers, etc
  • Main File - JIP MiniMap 1.50
  • Guide File - JIP Minimap Fonfig
  • Press N to toggle the map on & off; press-and-hold it to switch between local and world map modes
JIP Minimap Dynamic Colors Addon for JIP MiniMap mod with edited XML and frame texture to make it use HUD colors by default Main File - JIP MiniMap Dynamic Colors (New Icons Version) 1.0
Main & Pause Menus Overhaul Adds a bunch of features to the Main and Pause menus. Fully customizable
  • Main File - Main And Pause Menus Overhaul 3.0.1
  • Guide File - Main & Pause Menus Overhaul Fonfig
  • Press M from the pause menu to access the MCM
  • Press N from the pause menu to configure this mod's settings
Skill Book Checklist Adds an MCM checklist of the skill books in both wastelands & can optionally notify you when you enter a cell containing one Main File - Skill Book Checklist 2.0.3
Stash Organizer Automatically organizes your items in any interior cell, modded or vanilla
It is recommended to read the description for this mod so you understand how to use its features
  • Main File - Stash Organizer 2.1
  • Do NOT get the Optional File - Item Sorter 1.0, as we will be using ySI instead
  • Press the VATS key (default V) while the crosshair is on any container in any interior cell to get started
Vanilla HUD Cleaned Hand-made up-scaled/cleaned recreations of many vanilla HUD elements
If you use DarN UI, take extra care that you do NOT accidentally install the Vanilla UI+ Patch for this mod - this combination will lead to a crash
  • Main File - Vanilla Hud Cleaned
  • You will be prompted with a FOMOD installer
    • If you use VUI+, select everything under Modules. Leave the remaining options below unchecked.
    • If you use DarnUI, select all Modules EXCEPT:
      • Vanilla UI+ Patch
      • Clean Fonts
      Instead, check Clean Fonts for Darn
  • Clean Companion Wheel - Main File - Clean Companion Wheel 0.9
yUI - User Ynterface NVSE plugin that aims to fix UI bugs and add new UI features
  • Main File - yUI 1.4d
  • If you use VUI+: Matched Cursor - Fallout New Vegas 1.2
  • If you use DarnUI: Optional File - Matched Cursor - DarN UI 1.2
  • Guide File - yUI Fonfig
ySI - Sorting Ycons
  • Sorting mod using features of yUI to make inventory management a better experience
  • Each item gets assigned one of the 140+ included custom icons, representing different categories of items
 Main File - ySI - Sorting Icons 1.3
Requires bSortingIcons = 1 in yUI.ini.
It has been enabled for you in the yUI Fonfig provided above
