HUD/UI

Mod Name Description
The Mod Configuration Menu a.k.a. MCM. Required for many mods. You also need MCM BugFix 2 from the optional files!
Universal Interface Organizer a.k.a. UIO. Required for UI mods to coexist.
Vanilla UI Plus Modernized vanilla-style UI with many enhancements.
Darnified UI Alternative, smaller UI with extra HUD elements.
Vanilla HUD Cleaned High-quality HUD (use with VUI+).
Consistent Pip-Boy Icons High-quality icons that are used for perks, items, etc.
JIP Minimap Customizable realtime minimap that appears on the HUD.
