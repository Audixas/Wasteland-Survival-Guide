Essential Utilities

First, create a "Utilities" separator in MO2.

As you install the mods from this section, organize them underneath this separator.

All of the mods in this section are absolutely essential for TTW to function! Many mods in the following sections are optional, but make sure to get every Utility listed - they are all hard requirements.


Mod Name Description Instructions
xNVSE
New Vegas Script Extender 		Script extender. Required by the majority of mods. Unzip the contents of the archive to the Root folder, where FalloutNV.exe is located. Video instructions are also provided on the linked page.
JIP LN NVSE Script extender plugin; required by many mods. Normal install through MO2.
JohnnyGuitar NVSE Script extender plugin; required by many mods. Normal install through MO2.
ShowOff NVSE Script extender plugin; required for many mods. Download ShowOffNVSE.7z from the latest release and install manually through MO2. If you don't know what a .7z file is, install 7-Zip.
Heap Replacer Replaces the game's default memory management with a more efficient version, resulting in faster load times, reduced stutter, increased performance and stability. Instructions for Heap go here :snig:
kNVSE Animation Plugin Allows for custom animations to be loaded without overwriting, making it possible to use animations made specifically for certain weapons without affecting the others. Normal install through MO2.
Mod Limit Fix Greatly increases the plugin limit by fixing a vanilla bug. Improves performance, even if you are nowhere near the limit. Normal install through MO2.
New Vegas Tick Fix
(NVTF) 		Allows New Vegas to run above 60fps without physics issues, while reducing stuttering and increasing performance.
Download our NVTF Custom INI to automatically set up the mod. Further adjustments can be made if necessary however the provided INI should work fine for most users.
OneTweak Allows the game to run in borderless windowed mode, which allows you to Alt-Tab without the game locking up. Manual download from Skyrim Nexus, then install through MO2.
That's not a typo, the version on Skyrim Nexus is more up-to-date and will work perfectly fine with FNV. They are the same program, but this version never made it onto NV Nexus. Don't worry about it!
User Interface Organizer (UIO) Overcomes several limitations present in the vanilla console. Normal install through MO2.
