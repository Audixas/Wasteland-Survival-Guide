Wasteland Survival Guide

Essential Utilities

First, create a "Utilities" separator in MO2.

As you install the mods from this section, organize them underneath this separator.


All of the mods in this section are absolutely essential for TTW to function! Many mods in the following sections are optional, but make sure to get every Utility listed - they are all hard requirements.

Mod Name Description Instructions
4GB Patch
Makes Fallout New Vegas 4GB-aware & automatically loads NVSE.
  1. Download the Main File - FNV 4GB Patch 1.5
  2. From the downloaded archive, extract the .exe file to the game's Root folder
  3. Right-click on FNVpatch.exe and select Run as administrator
  4. A command prompt window will open and should say FalloutNV.exe patched!
  5. Close the command prompt and a file named FalloutNV_backup.exe should appear in the game's Root folder
xNVSE
New Vegas Script Extender 		Script extender. Required by the majority of mods including TTW.
  • Unzip the contents of the nvse_6_2_5.7z archive to the Root folder.
  • Video instructions are also provided on the linked page.
JIP LN NVSE
jazzisparis + Lutana 		Script extender plugin; required by TTW & many other mods. Main File - JIP LN NVSE Plugin 56.50
JohnnyGuitar NVSE Script extender plugin; required by many mods. Main File - JohnnyGuitar NVSE 4.55
kNVSE Animation Plugin Allows for custom animations to be loaded without overwriting, making it possible to use animations made specifically for certain weapons without affecting the others. Main File - kNVSE 20
Mod Limit Fix Greatly increases the plugin limit by fixing a vanilla bug. Improves performance, even if you are nowhere near the limit. Main File - FNVModLimitFix 2.5
KKNV (NVTF)
Korekcija kvachica u Novom Vegasu
(New Vegas Tick Fix) 		Allows New Vegas to run above 60fps without physics issues, while reducing stuttering and increasing performance. Main File - New Vegas Tick Fix 9.9.1i
The mod name is in Croatian right now. Do not panic if you are from a place where you panic when you see a foreign language.
NVHR
New Vegas Heap Replacer 		Replaces the game's default memory management with a more efficient version, resulting in faster load times, reduced stutter, increased performance and stability.

  • Download the Main File - NVHR 3.2 using the Manual Download button
  • From the downloaded archive, run the cpu_info.exe
  • A command prompt will tell you which version of the mod to use
Pay attention to the line that says => Use <=. For me it says AVX2, but it may be different for you.
  • Back in the downloaded archive, open the folder's which name corresponds to the version that the command prompt told you to use
  • From that folder, extract the d3dx9_38.dll to the game's Root folder
OneTweak Allows the game to run in borderless windowed mode, which allows you to Alt-Tab without the game locking up. Normal install through MO2. If you don't know how to install a mod that doesn't have a Nexus link, re-read the end of the MO2 section of the guide.
ShowOff NVSE Script extender plugin; required for many mods. Download ShowOffNVSE.7z from the latest release and install manually through MO2. If you don't know what a .7z file is, install 7-Zip.

Testing Utilities

In some rare cases, Mod Organizer 2 has issues handling .DLL files and they won't load in-game. If you can not get any of the mods in this section to work per the following instructions, then you should try installing the mods directly into the game's Data folder rather than through MO2. The files from the mods should end up in Fallout New Vegas\Data\NVSE\plugins).
  1. Launch the game using the Tale of Two Wastelands option in MO2 (NOT the NVSE option)
  2. Once the game has reached the main menu, hit the ~ key (above the tab key) to open the console
  3. Enter GetNVSEVersion, and the console should print NVSE version: 6
    • If not, xNVSE is installed incorrectly
  4. Next, enter GetIsLAA, and the console should print GetIsLAA >> 2
    • If it prints >> 0, then the 4GB Patch was applied incorrectly
    • If it prints >> 1, then you need to run Mod Organizer 2 as an administrator
    • If it prints an error message, then JIP LN NVSE is installed incorrectly
  5. Next, enter IsDLLLoaded d3dx9_38, and the console should print IsDLLLoaded "d3dx9_38.dll" >> 1
    • If it prints >> 0, then Heap Replacer is installed incorrectly
    • If it prints an error message, then JohnnyGuitar NVSE is installed incorrectly
  6. Next, enter IsDLLLoaded NVAC, and the console should print IsDLLLoaded "nvac.dll" >> 1
    • If not, then NVAC is installed incorrectly
  7. Next, enter IsDLLLoaded NVTF, and the console should print IsDLLLoaded "nvtf.dll" >> 1
    • If not, then NVTF is installed incorrectly
  8. Next, enter IsDLLLoaded mod_limit_fix, and the console should print IsDLLLoaded "mod_limit_fix.dll" >> 1
    • If not, then Mod Limit Fix is installed incorrectly
