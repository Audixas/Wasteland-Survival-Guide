Wasteland Survival Guide

Final Steps

W.S.G. Load Order

  1. Download the file manually
  2. Once the download has finished, click the button at the top of MO2 and select Open Profile folder
  3. Extract the loadorder.txt from the downloaded archive to opened folder and overwrite when prompted
  4. Once the file is extracted, right-click in the left pane of MO2 and select All Mods -> Refresh
  5. If everything went correctly, the final plugin in MO2's right pane should be Simple Natural Weathers for TTW.esp

Fonfig Separator

  1. Create one last separator in MO2 and name it "Fonfig" (or "Config" if you insist)
  2. In the "Filter" box located at the bottom-right corner of the left pane, type "fonfig" to filter the modlist
  3. Quite a few Fonfig files - belonging to several different separators - should appear in the left pane
  4. Hold the Control key and while keeping it held click each fonfig mod - NOT the separators - until they are all selected
  5. With all of the "Fonfig" files now selected, right click any one of them, then click Send to... => Separator... as pictured below
  1. Select the Fonfig separator you created in step 1 and you will see the left pane update accordingly - all fonfig files are now together under the new separator
  2. Click to restore the normal view of the left pane
While this process was not strictly necessary, it makes it easier to configure the mods in the guide from one convenient location, and also teaches you a bit more about how to use the MO2 interface

Controls

A few of the mods in the guide add hotkeys. Let's review them below:

These controls are just the guide defaults, but you can customize all of them in the Mod Configuration Menu or Tweaks Menu
Realtime Pause Menu
  • Stewie's Tweaks
    • M to open the map
    • J to open the quest journal
  • JIP Companions Command & Control
    • C to open the companion menu
    • Mousewheel to scroll through options
    • Middle mouse button to activate options
  • JIP Minimap:
    • N to enable/disable the minimap
    • Hold N to switch between local and world map modes
  • Stash Organizer:
    • VATS key on an interior container
  • Main & Pause Menu Overhaul:
    • M to show/hide the MCM
    • N to configure Main & Pause Menu Overhaul
