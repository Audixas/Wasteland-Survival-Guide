Wasteland Survival Guide

Final Steps

W.S.G. Load Order

  1. Download the mod manually
  2. Once the download has finished, click the button at the top of MO2 and select Open Profile folder
  3. Extract the loadorder.txt from the downloaded archive to opened folder and overwrite when prompted
  4. Once the file is extracted, right-click in the left pane of MO2 and select All Mods -> Refresh
  5. If everything went correctly, the final plugin in MO2's right pane should be DNWeathers.esp

Consistent Mod Icons

  • Download Consistent Pip-Boy Icons - Main File - 4. CPI - Mod Patches 4 and install it normally in MO2.
  • When prompted with the FOMOD installer, just leave everything checked
  • We are mostly doing this for Cyberware, but having icons for mods you don't use will not be harmful
We install this file at the end to ensure it overwrites all the other mods in the guide and we can see all of its icons

Controls

A few of the mods in the guide add hotkeys. Let's review them below:

Realtime Pause Menu
  • Stewie's Tweaks
    • M to open the map
    • J to open the quest journal
  • Loot Assist
    • C to auto-loot
    • T to tag items & recipes
  • JIP Minimap:
    • N to enable/disable the minimap
    • Hold N to switch between local and world map modes
  • Stash Organizer:
    • VATS key on an interior container
  • Main & Pause Menu Overhaul:
    • M to show/hide the MCM
    • N to configure Main & Pause Menu Overhaul
