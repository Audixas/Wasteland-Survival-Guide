Wasteland Survival Guide

Bugfix & Quality-of-Life

First, create a "Bugfix & QoL" separator in MO2.

As you install the mods from this section, organize them underneath this separator.


Mod Name Description Instructions
Casino Exchange All Trade in all Caps, Legion, NCR & (optionally) Pre-war Money at once, at the casino cashiers
  • Main File - Casino Exchange All 1-5a
    • When the FOMOD asks if you want to be able to exchange pre-war money, choose whichever option you want
  • Optional File - Voices for Casino Exchange All
Console Paste Support Adds clipboard paste support, as well as ctrl left/right and backspace/delete to aid typing in console Main File - Console Paste
FOV Slider Allows persistent field-of-view adjustments via the MCM
  • Main File - FOV Slider
  • Configure in the MCM to your preference
HUD-Only Explosion Shake Removes the godawful camera shake when hit by an explosion, but keeps the HUD shake so there is some sense of being near explosion Main File - HUD-Only Explosion Shake
Improved Console (NVSE) Overcomes several limitations present in the vanilla console Main File - Improved Console
JIP Improved Recipe Menu Essential convenience fixes for crafting and reloading workbenches Main File - JIP Improved Recipe Menu (Scriptrunner)
No More Motion Sickness Removes any kind of blur (motion blur, get hit blur, etc...) from the game Main File - No More Motion Sickness (No Plugin)
No More Explosion Blindness Removes the annoying blindness effect from explosions Main File - No More Explosions Blindness (No Plugin)
No Muzzle Flash Lights Improves game performance in combat by disabling lights cast on the environment by muzzle flashes, which are very performance heavy in New Vegas and Fallout 3
It doesn't disable the muzzle flashes themselves; only the radiant light cast by the flashes
 Main File - New Vegas
If you don't know how to install a mod that isn't from Nexus, refer to the Mod Installation Advice at the end of the MO2 section of the guide
Tutorial Killer Removes message boxes that pop up during the tutorial, as well as those that occur once during a new game, and repeating corner-messages
  • Main File - Tutorial Killer
  • Guide File - Tutorial Killer Fonfig
    • This extra file makes DLC silent instead of corner messages, silences the MCM & more
