Wasteland Survival Guide

Bugfix & Quality-of-Life

First, create a "Bugfix & QoL" separator in MO2.

As you install the mods from this section, organize them underneath this separator.


Mod Name Description Instructions
lStewieAl's Tweaks
  • Engine bugfixes, optional tweaks and new features with no performance impact
  • Fully customisable via in-game menu and INIs
  • Literally hundreds of features
  • The biggest and baddest mod there is
 Normal install through MO2. Requires an INI to function, provided below.
Configured via Settings/Tweaks in the title/pause menu. Changes made from this in-game menu require a game restart to take effect.
Download the Wasteland Surival Guide Tweaks INI and install it as its own mod in MO2. This way, your INI customizations will survive when the main mod receives an update.
It can't be overstated just how many different things this mod does. It renders countless older mods obsolete with its more performance-friendly structure, while also providing a gorgeous in-game menu that you can browse by category or search by keyword. This makes it easy to customize your game just the way you want it. Familiarize yourself with this menu and make a habit of checking it before attempting to install a mod. More often than not you may find that an equivalent tweak already exists. Stewie's Tweaks is one of the most versatile and powerful tools a modder has at their disposal, without question.
FOV Slider Allows persistent field-of-view adjustments via the MCM. Normal MO2 installation. Configure in MCM.
JIP Improved Recipe Menu Essential convenience fixes for crafting and reloading workbenches. Normal MO2 installation. Nothing to configure.
Stash Organizer Automatically organizes your items in any interior cell, modded or vanilla. Normal MO2 installation. Press VATS key while the crosshair is on any container in any interior cell to get started.
Tutorial Killer Bordering on self-explanatory, this mod disables irritating tutorial messages. Normal MO2 installation. Nothing to configure, however suppressing additional tutorials by adding more .ini files is supported.
