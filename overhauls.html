Wasteland Survival Guide

Content

First, create an "Overhauls" separator in MO2.

As you install the mods from this section, organize them underneath this separator.
The mods in this section mostly increase the difficulty of the game. They aren't as recommended for a first playthrough and you should feel free to use only the ones you want!
Mod Name Description Instructions
BLEED
Ballistic Locational Energy
Explosive Damage
  • Overhauls how damage and force is applied
  • Makes combat much more unforgiving and closer to classic Fallout
  • Crippled limbs on crit, burning and bleeding effects and more
 Main File - BLEED 1.8
BLEEDLESS
A lighter version of RoyBatty's excellent BLEED; it retains most changes relating to damage while removing its more intrusive changes
This is to be used instead of BLEED, not alongside it.
  • Main File - BLEEDLESS 1.0
  • Optional File - Better Damage Multipliers 1.0
Benny Humbles You
And Steals Your Stuff
  • A subsidiary variant of the original Benny Humbles You
  • Resets the player's Level, XP, Skills, Perks and Karma upon travelling from the CW to the Mojave for the first time
  • Takes all the player's gear, locks it in a safe in the Tops Presidential Suite, and gives Benny the key
  • This is done to preserve the balance of the early game in New Vegas
  • Highly customizable in the INI if you would like to decide what to reset and what to keep
  • Main File - Benny Humbles You and Steals Your Stuff 7
  • Nail Gun Semi Auto Rework - Main File - Nail Gun Semi Auto 1fix
Charisma Tweaks
(NVSE) 		This mod makes it so barter and speech checks (and optionally, ALL skill checks) take into account both your charisma and the intelligence of the person you're speaking to (optionally, it can use the charisma of the person you're speaking to instead). Main File - Charisma Tweaks 0.3b
Follower Formula Redone Changes the amount of followers for you can have to your Charisma stat/2, rounded down. Charisma is no longer a dump stat!
  • Main File - Follower Formula Redone
  • Patch - FFR - TTW 1.0
GRA Unique Weapons Relocated
GRA Scavenger Hunt
Makes the GRA unique weapons unavailable for purchase and scatters them across the Mojave, Zion, and the Big MT - just like all the other unique weapons.
  • Main File - GUWR Unbalanced 1.51
  • Patch - gra scavenger hunt unbalanced - ttw patch
Harder New Vegas Strip Access Makes it harder to access The Strip in New Vegas, forcing the player to engage with the early game - especially if Benny Has Stolen their Stuff. Main File - Charisma Tweaks 0.3b
Infirmary Overhaul TTW Changes the infirmary. No limitless healing, rad removing or limb fixing. Your skill in medicine will greatly affect the amount healed or rads removed Main File - Infirmary Overhaul TTW 1
JIP LN Fonfig Customizes some settings in JIP LN to make the game more dynamic and challenging, for example by distributing weapon mods to NPCs. Main File - JIP LN Fonfig
Misc Gamesetting Tweaks Customizes some vanilla gamesettings with text files that are powered by Stewie's Tweaks We suggest:
  • asdf
Save Restrictions Save Restrictions comes with the following optional features, configurable in its ini:
  • Save only while sitting: It makes it so you can only save if you're sitting in a chair/bench/whatever
  • Save only while not in combat: It makes it so you cannot save while in [DANGER]
  • Save only in interiors: Makes it so you cannot save in the middle of the desert
  • Disable the quickload hotkey: Just so you don't spam it every second
 Main File - Save Restrictions 0.1
SawyerBatty
  • Mostly based on the famous JSawyer mod by FNV's director Josh Sawyer
  • Edited by RoyBatty to revert the carry weight, level cap & Logan's Loophole change, fix many bugs, and for parity with TTW
  • The pre-order packs are placed in logical areas around Springvale and Megaton when you start in DC
 Main File - SawyerBatty 1.2
Simple Diseases A simple and modular disease system for Fallout New Vegas. Adds a chance to get sick from eating bad food and drinking untreated water as well as the risk of contracting diseases from wild animal attacks. Main File - Simple Diseases 1.3
Three Perk Bounty
  • Upon level up you will be presented with a choice of three randomly selected perks from the available perk pool
  • You will also have an option to re-roll the random selection up to three times...
  • ...however, this carries a chance of giving you an unlucky perk active until next level (Luck over 5 improves your chances a bit)
TTW Quick Start Quick start plugin to skip Vault 101 and start in the cave just outside the vault. The Medicine Bobblehead is not given at the start, you can pick it up later during Trouble on the Homefront Main File - TTW Quick Start 1.3
Vault Starting Gear DC Changes what gear you're given by Amata at the start of Escape!. Effort has been made to make sure that the starting gear could plausibly have been found in Vault 101 over the last few years. TTW Quickstart compatible! Main File - Vault Gear DC Quickstart Version 3.3fix
Weapon Requirements System
Makes weapon Strength and Skill requirements matter, allowing you to use any weapon but assigning significant dynamic penalties for those you aren't proficient with Main File - Weapon Requirements System 1.0.4
