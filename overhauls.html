Wasteland Survival Guide

Overhauls

First, create an "Overhauls" separator in MO2.

As you install the mods from this section, organize them underneath this separator.
Some of the mods in this section are extremely spicy. While they are all compatible with each other, you are not expected to use every single one of them simultaneously; you might even be slightly deranged if you try that. This section intentionally gives you more options than you should realistically need on the same playthrough so that you can design your own level of challenge. There is no special achievement for making the game way too difficult for you to enjoy, so use the mods you think sound fun for you and don't over-do it!
Mod Name Description Instructions
(Benny Humbles You) and Steals Your Stuff
  • A subsidiary variant of the original Benny Humbles You, aimed at preserving the balance of the early game in New Vegas
  • Customizable reset of Level, XP, Skills, Karma upon travelling from the CW to the Mojave for the first time
  • Takes all the player's gear, locks it in a safe in the Tops Presidential Suite, and gives Benny the key
Skip this mod if you plan to start in the Mojave instead of DC. Not only does the premise of the mod not make sense playing in that order, moreover you will experience character creation issues at Doc Mitchell's. This mod is only for DC start, which is the (soft) recommendation for TTW
  • Main File - Benny Humbles You and Steals Your Stuff 9update2
  • Nail Gun Semi Auto Rework - Main File - Nail Gun Semi Auto 1fix
  • Double-click (Benny Humbles You) and Steals Your Stuff in the left pane, choose INI Files & decide what to reset and what to keep
BLEED
Ballistic Locational Energy Explosive Damage
  • Overhauls how damage and force is applied
  • Makes combat much more unforgiving and closer to classic Fallout
  • Crippled limbs on crit, burning and bleeding effects and more
  • Main File - BLEED 1.8.5
  • Ragdolls users only - ttw - ragdolls - bleed patch 1.1
BLEEDLESS A lighter version of RoyBatty's excellent BLEED; it retains most changes relating to damage while removing its more intrusive changes
This is to be used instead of BLEED, not alongside it
  • Main File - BLEEDLESS 1.1
  • Optional File - Better Damage Multipliers 1.0
    • This standalone optional will also work with the original BLEED. Recommended!
  • Ragdolls users only - Ragdolls - BLEEDLESS Patch 1.0
Charisma Tweaks (NVSE) This mod makes it so barter and speech checks (and optionally, ALL skill checks) take into account both your charisma and the intelligence of the person you're speaking to (optionally, it can use the charisma of the person you're speaking to instead) Main File - Charisma Tweaks 0.3b
Famine - A Loot Rarity Mod A simple and comprehensive loot scarcity mod using event-based scripting
  • Main File - Famine 1.2.5
  • Optional File - Famine - Flora TTW Patch 1.2.4
GRA Unique Weapons Relocated
GRA Scavenger Hunt
Makes the GRA unique weapons unavailable for purchase and scatters them across the Mojave, Zion, and the Big MT - just like all the other unique weapons
  • Main File - GUWR Unbalanced 1.51
  • Patch - gra uniques relocated unbalanced - ttw patch 1.0
Harder New Vegas Strip Access Makes it harder to access The Strip in New Vegas, forcing the player to engage with the early game - especially if Benny has Stolen their Stuff Main File - Harder New Vegas Strip Access 2.0
Infirmary Overhaul TTW Changes the infirmary. No limitless healing, rad removing or limb fixing. Your skill in medicine will greatly affect the amount healed or rads removed Main File - Infirmary Overhaul TTW 2
JIP LN Fonfig Customizes some settings in JIP LN to make the game more dynamic and challenging, for example by distributing weapon mods to NPCs Click the mod name to download the main file and install in MO2.
Player Combat Priority Prevents the game becoming too easy by making enemies more likely to target the player in combat rather than companions Main File - Player Combat Priority 1.0.2
RAD - Radiation (is) Actually Dangerous Radiation now damages your health at a rate of 1% HP per 10 rads until it is removed; the UI reflects the affected portion in red like in Fallout 4. With hardcore mode enabled, your primary needs will decrease your AP as well; dehydration penalties are twice as severe
  • Main File - RAD - Overhaul 2.1.0
  • Guide File - RAD Fonfig
ROOG - Roy's Overhaul Of Gameplay
    Restores Fallout 3 weapon degrading, repair and overhauls the difficulty of the game for a much harder experience and slower progression
    This is the mother of all challenge mods. Make sure you read the description thoroughly and understand what you are getting yourself into. This will dramatically overhaul the difficulty & pacing of the game, and is not configurable
  • Main File - ROOG 1.8
  • Optional File - sawyerbattyROOG 1.2
Save Restrictions Save Restrictions comes with the following optional features, configurable in its ini:
  • Save only while sitting in a chair/bench/whatever
  • Save only while not in combat i.e. [DANGER]
  • Save only in interiors
  • Disable the quickload hotkey
  • Main File - Save Restrictions 0.1
  • Double-click the mod in the left pane & select INI Files
  • Select the Config\SaveRestrictions\SR.ini file in the left pane
  • Set your desired restrictions in the right pane
SawyerBatty
  • Mostly based on the famous JSawyer mod by FNV's director Josh Sawyer
  • Edited by RoyBatty to revert the carry weight, level cap & Logan's Loophole change, fix many bugs, and for parity with TTW
  • The pre-order packs are placed in logical areas around Springvale and Megaton when you start in DC
 Main File - SawyerBatty 1.12
Simple Diseases A simple and modular disease system for Fallout New Vegas. Adds a chance to get sick from eating bad food and drinking untreated water as well as the risk of contracting diseases from wild animal attacks Main File - Simple Diseases 1.4
Stewie Tweaks Overhaul Fonfig Enables more advanced gameplay and challenge-enhancing tweaks which are left out of the starter INI for being too spicy Install this as its own separate mod in MO2 below the other INI
Three Perk Bounty
  • Upon level up you will be presented with a choice of three randomly selected perks from the available perk pool
  • You will also have an option to re-roll the random selection up to three times...
  • ...however, this carries a chance of giving you an unlucky perk active until next level (Luck over 5 improves your chances a bit)
 Click Download⌗ and install in MO2
TTW Hardcore Zeta Moves your gear container to after the space walk, which makes Zeta more challenging and encourages exploration and using the equipment you find
  • Main File - TTW Hardcore Zeta 1.2
  • Patch File - TTW Hardcore Zeta - Delay DLC Redux Patch 1.2
TTW Quick Start Quick start plugin to skip Vault 101 and start in the cave just outside the vault. The Medicine Bobblehead is not given at the start, you can pick it up later during Trouble on the Homefront
  • Main File - TTW Quick Start 1.4
  • TTW Quick Start Item Remover - Main File - TTW Quick Start Item Remover 2.0
Wasteland Starting Gear DC Provides you with stat-based starting gear at the beginning of your TTW playthrough. A staged mock battle near Vault 101's exit door adds the gear to the dead combatants for you to loot Main File - Wasteland Starting Gear DC 1fix2
Weapon Requirements System
Makes weapon Strength and Skill requirements matter, allowing you to use any weapon but assigning significant dynamic penalties for those you aren't proficient with Main File - Weapon Requirements System 1.0.4
