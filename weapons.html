Weapons & Animations

First, create a "Weapons & Animations" separator in MO2.

As you install the mods from this section, organize them underneath this separator.

TTW already includes a lot of weapons, but consider these optional mods if you'd like to add even more to your arsenal!
You can spruce things up with some fancy animations too thanks to kNVSE.

Mod Name Description Instructions
Weapons of the Modern Millenia Wepon! Normal install through MO2. If you installed Anniversary Animation Pack, get the patch for that mod as well from the Optional Files section.
You will also probably need something from the Vanilla Flavoured Patches page that is linked to in the description. The file you should choose depends on which mods you are using together, so read the description very carefully and choose the appropriate file(s).
