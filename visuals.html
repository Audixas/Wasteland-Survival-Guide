Wasteland Survival Guide

Visuals

First, create a "Visuals" separator in MO2.

As you install the mods from this section, organize them underneath this separator.

You may notice this guide does not recommend any texture packs. This is because they are shit.

Mod Name Description Instructions
Improved LOD Noise Texture Small but nice improvement to distant land texture which can otherwise be more jarring. Normal install through MO2.

Weather Mods - Don't try to use all of these at once, read carefully!

Mod Name Description Instructions
Nevada Skies Complex and heavily-scripted weather mod featuring "400+ weather types" and extensive customization via MCM.
Choose this weather mod if you want intense, vivid, high-fantasy weathers in your game and you have a high tolerance for fiddling with settings in the MCM. If Altitude is the minimalist weather mod, Nevada Skies is the maximalist, with all the pros and cons that brings.
Check out these Example Screenshots courtesy of Voidwalker.
 Normal install through MO2, then install at minimum the interior lighting fix.
You need the SILO Integration file from Nevada Skies Tweaks by Qolore7. The other files are also highly recommended, but are somewhat more subjective.
