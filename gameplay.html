Wasteland Survival Guide

Gameplay

First, create a "Gameplay" separator in MO2.

As you install the mods from this section, organize them underneath this separator.

Mod Name Description Instructions
Baka Skill Book Scanner TTW Dynamic quest markers that highlight nearby skill books, anywhere, anytime Main File - Baka Skill Book Scanner TTW
Better Hacking Bypass the hacking mini-game by using a Password Bypass, a new item found alongside bobby pins. If you don't have any bypasses, you can still do the vanilla mini-game. You still need the required Science levels Main File - Better Hacking
Blackout Nightvision Night vision effect buffs perception in interiors or night time, but nerfs charisma during the daytime because you look like a dweeb Main File - immersive ultra trealtics niaght fision too
Bottle Rinse Repeat Fill water bottles at any water source. Also overhauls the Vault 13 canteen & allows you to milk brahmin because why not
  • Main File - Bottle Rinse Repeat
  • Optional File - Bottle Rinse Repeat - TTW Patch
BURN - Hardcore Fire Effects Causes unprotected humans and flammable creatures to panic and flee when set on fire, adding an extra gameplay mechanic for incendiary weapons and ammo
This mod does restore some companion screams cut from FNV which some people find disturbing. Give this one a skip if realistic screams of pain would be a trigger for you
  • Main File - BURN - Hardcore Fire Effects
  • Optional File - BURN - TTW Patch (Script Runner)
Canvas Backpacks - Remade Provides backpack resources which we will distribute throughout the game using the following mod, CIBS
The reason we are hiding the plugin in the installation steps is that we will use CIBS to replace its functionality
  • Main File - Canvas Backpacks - Remade
  • Optional File - Strapless Update
  • Once the mod has been installed, double-click it in the left pane of MO2 and select Optional ESPs
  • Select QuibNewBackpacks.esp and click the green to move it from Available ESPs into Optional ESPs
CIBS - Customizable Integrated Backpack System Adds an integrated backpack system into New Vegas. Completely customizable via INI, & provides a framework for adding backpacks, bags, or armors from other mods without having to use the plugins from said mods Main File - CIBS - Customizable Integrated Backpack System
Cyberware 2281 Recreates the functionality of the Cyberware implants from Project Nevada, without all the problems caused by that ancient mod
  • Main File - Cyberware 2281
  • Optional File - Cyberware 2281 TTW Patch
JAM - Just Assorted Mods
  • A compilation of interface and gameplay-enhancing mods in one package, configurable in MCM
  • HUD features like Dynamic Crosshair, Hit Marker, Hit Indicator, Visual Objectives
  • Gameplay features like FO4-style Loot Menu, Sprint, Bullet Time, Hold Breath, Weapon Hweel
  • Main File - Just Assorted Mods
  • Guide File - JAM Fonfig
  • JAM Sprint Animation Replacers
  • Configure the mod in the MCM if you wish, or disable features entirely if you don't plan to use them
CyberJAM
  • Integrates Cyberware 2281 and Just Assorted Mods
  • Allows them to interact like the old PN Core and Cyberware modules
  • Reintroduces and expands on the cut Wired Synapses implant
  • Main File - CyberJAM
  • Optional File - CyberJAM INI File
Delay DLC Redux TTW Prevents the game's DLC from spamming the new game startup messages and requires you be near a given DLC's starting location to receive its quest
It cannot be stressed enough that you need to read the description for this mod so you understand how to start the DLCs while it is installed
 Main File - Delay DLC Redux TTW
Essential DLC Enhancements Merged A collection of small essential gameplay improvements for the official DLCs that have been fully merged, updated, and cleaned Main File - Essential DLC Enhancements
Enhanced Vision
  • Power Armor helmets and other high-tech gear now enable enhanced vision modes like Night Vision, Heat Vision and EM Vision
  • If there's no Power Armor equipped as a power source, energy cells are required while these modes are active
  • Extracted from Project Nevada, optimized and refactored
 Main File - Enhanced Vision
Immersive Recoil 2.0 Adds recoil to weapons in a stable and compatible fashion. Can be customized in MCM Main File - Immersive Recoil
Jamming Fix & Optional Tweaks Fixes the on-fire jamming for automatic weapons and adds option for how often weapons jam
  • Main File - Jamming Fix and Optional Tweaks
  • Guide File - JAM Fix Fonfig
JIP CC & C
Companions Command and Control
  • Revolutionizes the way companions are controlled and interacted with
  • A must-have, it has every feature imaginable when it comes to companions
  • Main File - JIP Companions Command and Control
    • JIP CCC HD Icons is also recommended
    • If you don't like the avatars, try Vault Boy JIP CCC Avatars
    • Another great avatars option is Perk Styled CCC Icons
Misc Gamesetting Tweaks Customizes some vanilla gamesettings with text files that are powered by Stewie's Tweaks
You can either use the WSG Recommended Settings, or customize your own. Make sure you remember what you picked before reporting "bugs" like jumping too high. Reinstall the mod at any time to change your selections
  • Main File - Misc Gamesetting Tweaks Presets
  • In the FOMOD installer just click
    • If you picked Perk Every Level and plan to play on Hardcore mode, install TTW Hardcore Perk Every Level (Pluginless)
NPCs Sprint in Combat NPCs will now sprint in melee combat instead of casually jogging. Uses custom sprint animations. Aims to make combat more scary and realistic Main File - NPCs Sprint in Combat
Ragdolls TTW Improves ragdoll behaviour for all NPCs & creatures in the game & allows hit reactions, forces, momentum & more to be configured in the MCM
This mod requires New Vegas Compatibility Skeleton, which we install later during the "Aesthetics" section
  • Main File - Ragdolls TTW
  • Patch - Ragdolls TTW 3.3 Patch
Reload Reloaded Replacements for agility modifiers to reload and draw speed, strength modifiers to throwing range, and custom sneak attack damage multipliers Main File - Reload Reloaded
Root 'n Loot TTW Replaces all static containers with lootable equivalents Main File - Root'n Loot
Stash Organizer Automatically organizes your items in any interior cell, modded or vanilla
It is recommended to read the description for this mod so you understand how to use its features
  • Main File - Stash Organizer
  • Do NOT get the Optional File - Item Sorter, as we will be using ySI instead
  • Press the VATS key (default V) while the crosshair is on any container in any interior cell to get started
Supplemental Ammo Crafting Adds crafting and breakdown recipes for pretty much all of the ballistic ammo types that were missing them in TTW Main File - Supplemental Ammo Crafting
Sweet 6 Shooter Perks - TTW Adds 60+ perks and 12+ traits in a vanilla friendly, immersive way. Featuring specialization perks for revolvers, bolt-action weapons, wasteland/Mad Max themed weapons, rockets, communist/guerilla weapons, handguns and more, this perk pack aims to flesh out the perk selection throughout the game for just about every playstyle, with a heavy focus on proper balance and role-playing Main File - Sweet 6 Shooter Perk Pack TTW Version
Tale of Bi Wastelands Adds Confirmed Bachelor and Cherchez La Femme options for the canonically gay & lesbian merchants to mirror any Black Widow and Lady Killer options in Fallout 3 and its DLC Main File - Tale of Bi Wastelands
TTW - Mothership Zeta Rewards Adds a container upon completing Mothership Zeta, it will be populated with the items from any abductees you decided to keep alive during your playthrough. Think Honest Hearts, but better! Main File - TTW - Mothership Zeta Rewards
TTW Reputations Adds a functional reputation system to Fallout 3 for parity with New Vegas; compatible with TTW New Vegas Speech Checks Main File - TTW Reputations
TTW New Vegas Speech Checks Converts all Fallout 3 dialogue to use the New Vegas speech system; compatible with TTW Reputations
  • Main File - TTW New Vegas Speech Checks
  • Optional File - TTW New Vegas Speech Checks - TTW Reputations Patch
  • TTW NV Speech Checks & Reputations - Tale of Bi Wastelands Patch
TTW Transportalponder Extended Upgrades the vanilla Big MT Transportalponder! received at the conclusion of Old World Blues into a configurable teleportation device. Over 70 destinations are available, with options to expand to over 100 destinations, or reduce to around 20. Custom recall points are also possible, allowing quick access to The Sink and back! Main File - TTW Transportalponder Extended
Wild Wasteland Prompt Turns Wild Wasteland into a choice made from a dialog prompt, as opposed to a trait. Main File - Wild Wasteland Prompt
