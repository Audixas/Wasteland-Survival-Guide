Wasteland Survival Guide

Gameplay

First, create a "Gameplay" separator in MO2.

As you install the mods from this section, organize them underneath this separator.

The mods in this section are optional, but highly recommended to enhance the game. Feel free to leave out any that aren't your style!

Mod Name Description Instructions
xNVSE Script extender. You need this! Unzip the contents of the archive to the root folder, where FalloutNV.exe is located.
JIP LN NVSE Script extender. You need this! Normall install through MO2.
JohnnyGuitar NVSE Script extender. You need this! Normall install through MO2.
Mod Limit Fix Bugfix which greatly increases the plugin limit and improves performance, even if you are nowhere near that limit. Normall install through MO2.
