Wasteland Survival Guide

Gameplay

First, create a "Gameplay" separator in MO2.

As you install the mods from this section, organize them underneath this separator.

The mods in this section are all very optional; feel free to leave out any that aren't your cup of tea.

Mod Name Description Instructions
BLEED
  • Overhauls how damage and force is applied
  • Makes combat much more unforgiving and closer to classic Fallout
  • Crippled limbs on crit, burning and bleeding effects and more
 Normal install through MO2.
JIP Companions Command and Control
  • Revolutionizes the way companions are controlled and interacted with
  • A must-have, it has every feature imaginable when it comes to companions
  • Normal install through MO2.
  • If you don't like the default style, try Vault Boy JIP CCC Avatars
  • Another great option is Perk Styled CCC Icons
TTW New Vegas Speech Checks
  • Converts all Fallout 3 dialogue to use the New Vegas speech system
  • Compatible with TTW Reputations
    Normal install through MO2.
    If you use TTW Reputations then get the Optional File - TTW New Vegas Speech Checks - TTW Reputations Patch
TTW Reputations
  • Adds a functional reputation system to Fallout 3 for parity with New Vegas
  • Compatible with TTW New Vegas Speech Checks
    Normal install through MO2.
    If you use TTW New Vegas Speech Checks, download Optional File - TTW New Vegas Speech Checks - TTW Reputations Patch from that mod's page
