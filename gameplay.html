Wasteland Survival Guide

Gameplay

First, create a "Gameplay" separator in MO2.

As you install the mods from this section, organize them underneath this separator.

Mod Name Description Instructions
Better Hacking Bypass the hacking mini-game by using a Password Bypass, a new item found alongside bobby pins. If you don't have any bypasses, you can still do the vanilla mini-game. You still need the required Science levels Main File - Better Hacking - 1.3
Bottle Rinse Repeat
    Fill bottles from any source. Point at a water source, the interaction prompt will show if your first pressing of “Activate” will fill some bottles. Once it's done, the prompt will change to vanilla action. If you don't have any empty bottles, upgradeable liquids, or caps to seal them, you will see the vanilla prompt as usual.
  • Main File - Bottle Rinse Repeat 3.0
  • Optional File - Bottle Rinse Repeat 3.0 - TTW Patch
Canvas Backpacks - Remade Provides backpack resources which we will distribute throughout the game using the following mod, CIBS
  • Main File - Canvas Backpacks - Remade v1
  • Optional File - Strapless Update v1
  • Once the mod has been installed, double-click it in the left pane of MO2 and select Optional ESPs
  • Select QuibNewBackpacks.esp and click the green to move it from Available ESPs into Optional ESPs
The reason we are hiding the plugin for this mod is that the following mod, CIBS, will provide the same functionality in a better way.
CIBS
Customizable Integrated Backpack System 		Adds an integrated backpack system into New Vegas. It is completely customizable, thanks to INI functions and JIP LN Script Runner. It also provides a framework for adding backpacks, bags, or armors from other mods, completely via script without the need for extra .esp files. Main File - CIBS - Customizable Integrated Backpack System -
No Script Runner version 3.2.0-beta
Cyberware 2281
Recreates the functionality of the Cyberware implants from Project Nevada, without all the problems caused by that old mod
  • Main File - Cyberware 2281 1.3.3
  • Optional File - Cyberware 2281 TTW Patch 1.3.3
JAM
Just Assorted Mods
  • A compilation of interface and gameplay-enhancing mods in one package, configurable in MCM
  • HUD features like Dynamic Crosshair, Hit Marker, Hit Indicator, Visual Objectives
  • Gameplay features like FO4-style Loot Menu, Sprint, Bullet Time, Hold Breath, Weapon Hweel
    		•
    • Main File - Just Assorted Mods 4.3d
    • Guide File - JAM Fonfig
    • Configure the mod in the MCM if you wish, or disable features entirely if you don't plan to use them
    CyberJAM
    • Integrates Cyberware 2281 and Just Assorted Mods
    • Allows them to interact like the old PN Core and Cyberware modules
    • Reintroduces and expands on the cut Wired Synapses implant
    • Main File - CyberJAM 1.01
    • Optional File - CyberJAM INI File
    Enhanced Vision
    • Power Armor helmets and other high-tech gear now enable enhanced vision modes like Night Vision, Heat Vision and EM Vision
    • If there's no Power Armor equipped as a power source, energy cells are required while these modes are active
    • Extracted from Project Nevada, optimized and refactored
    		 Main File - Enhanced Vision 1.3.1
    Immersive Recoil 2.0 Adds recoil to weapons. Built on the framework provided by the JIP LN NVSE Plugin to be the most stable and compatible recoil mod to date Main File - Immersive Recoil 2.3.2
    JIP CCC
    Companions Command and Control
    • Revolutionizes the way companions are controlled and interacted with
    • A must-have, it has every feature imaginable when it comes to companions
    • Main File - JIP Companions Command and Control 2.60
      • JIP CCC HD Icons is also recommended
      • If you don't like the avatars, try Vault Boy JIP CCC Avatars
      • Another great avatars option is Perk Styled CCC Icons

    Tale of Bi Wastelands
  • Adds Confirmed Bachelor and Cherchez La Femme options to mirror any Black Widow and Lady Killer options in Fallout 3 and its DLC
  • Adds new ones for the canonically gay and lesbian merchants, and an option to tell Dad
    		•  Main File - Tale of Bi Wastelands 1.0
    TTW Reputations
    • Adds a functional reputation system to Fallout 3 for parity with New Vegas
    • Compatible with TTW New Vegas Speech Checks
    		 Main File - TTW Reputations 1.1
    TTW New Vegas Speech Checks
    • Converts all Fallout 3 dialogue to use the New Vegas speech system
    • Compatible with TTW Reputations
    • Main File - TTW New Vegas Speech Checks 1.2
    • Optional File - TTW New Vegas Speech Checks - TTW Reputations Patch 1.2
      • Obviously, if you skipped TTW Reputations then you do not need this patch.
    Wild Wasteland Prompt Turns Wild Wasteland into a choice made from a dialog prompt, as opposed to a trait. Main File - Wild Wasteland Prompt 1.0
