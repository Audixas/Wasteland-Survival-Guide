Wasteland Survival Guide

Aesthetics

First, create a "Aesthetics" separator in MO2.

As you install the mods from this section, organize them underneath this separator.
If you are curious about/disappointed by the lack of texture mods in this section, the WSG team doesn't really use texture mods & they are generally compatible with everything, so for that reason they are considered outside of the scope of the guide. If you are interested in texture packs, here is a texture guide contributed by Salamand3r to help you get started. No support is provided for these textures, but do enjoy

SILO
Simple Interior Lighting Overhaul
  • Scripted simple solution for more immersive interiors. No FPS loss, no bloat
  • Automatic support for all interiors, including ones added by mods - including TTW
Main File - Simple Interior Lighting Overhaul 1.2
TTW Sun Tweaks Adjusts the sun glare for weathers in Tale of Two Wastelands, with parity across Fallout New Vegas values Main File - TTW Sun Tweaks 1.1
Altitude Vastly improves all the weathers in New Vegas, without stepping on the toes of the vanilla design too much Main File - Altitude 3.1
Simple Natural Weathers TTW New natural looking weathers for all of the Capital Wasteland, Anchorage, Point Lookout, and The Pitt Main File - Simple Natural Weathers 0.99.4
There are several other great weather mods for you to consider, including Desert Natural Weathers TTW (covers BOTH wastelands), Restored Weathers and IMODs (covers the Mojave) & TTW Vibrant Weathers (covers DC)
It is very important to note that you shouldn't use "overlapping" weather mods - make sure that the DC and the Mojave wastelands are only covered once each.
