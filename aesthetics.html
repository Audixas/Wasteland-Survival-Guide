Aesthetics

First, create a "Aesthetics" separator in MO2.

As you install the mods from this section, organize them underneath this separator.
Mod Name Description Instructions
Anniversary Anim Pack
  • Includes three of Hitman's previous animation/bugfix mods, Subtle Camera Motion, Iron Sights Recoil Animations, and Fire Animation Variants
  • Also includes some new previously unreleased animations
  • Download and install the Main File - Anniversary Anim Pack 1.9c through MO2
  • Once the mod has been installed, double-click it in the left pane of MO2 and select Filetree
  • Delete every folder EXCEPT the characters folder
Anniversary Anim Pack - General Bugfix
  • Includes three of Hitman's previous animation/bugfix mods, Subtle Camera Motion, Iron Sights Recoil Animations, and Fire Animation Variants
  • Also includes some new previously unreleased animations

  • Main File - Bonus Patch 1.3a
FNV Clean Animations
  • Includes three of Hitman's previous animation/bugfix mods, Subtle Camera Motion, Iron Sights Recoil Animations, and Fire Animation Variants
  • Also includes some new previously unreleased animations
  • Main File - FNV Clean Animations 2.2
  • Update File - FNV Clean Animations - Update 2.2.2
Configurable Pip-Boy Light Colour
  • Allows you to customize the light radius and RBG values of the Pip-Boy light

  • Includes the functionality of SmoothLight - Pip-Boy Light Enhancer
Configurable Pip-Boy Light Color 2.0
Improved LOD Noise Texture
  • Vastly improves the LOD noise texture used on all distant land

 Main File - Improved LOD noise Texture V1.00

  • Custom textures for all the power armors in FNV and F3/TTW

  • Guide File - PAVE TTW 3.3 Patch
