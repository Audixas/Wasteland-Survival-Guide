Wasteland Survival Guide

Aesthetics

First, create a "Aesthetics" separator in MO2.

As you install the mods from this section, organize them underneath this separator.

You may notice this guide does not recommend any texture packs. This is because they are shit.

Mod Name Description Instructions
Improved LOD Noise Texture
  • Vastly improves the LOD noise texture used on all distant land
  • Uses only minimal video memory and greatly improves quality
 Main File - Improved LOD noise Texture V1.00
PAVE TTW
Power Armor Visual Enhancement
  • Custom textures for all the power armors in FNV and F3/TTW
  • Also includes recon armor
  • Main File - Power Armor Visual Enhancement 2K 1.2
  • Update File - Power Armor Visual Enhancement Mesh Patch 1.0
SILO
Simple Interior Lighting Overhaul
  • Scripted simple solution for more immersive interiors. No FPS loss, no bloat
  • Automatic support for all interiors, including ones added by mods (including TTW
Main File - Simple Interior Lighting Overhaul 1.2
Configurable Pip-Boy Light Colour
  • Allows you to customize the light radius and RBG values of the Pip-Boy light
  • Settings in the INI file config/Configurable Pip-Boy Light Color.ini
  • Includes the functionality of SmoothLight - Pip-Boy Light Enhancer
Main File - Simple Interior Lighting Overhaul 1.2
