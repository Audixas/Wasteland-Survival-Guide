Wasteland Survival Guide

Aesthetics

First, create a "Aesthetics" separator in MO2.

As you install the mods from this section, organize them underneath this separator.
Weapon Animations
iscontroller REQUIRED for the following animations - automatically creates ##ISControl nodes for weapons with sightingnodes Main File - iscontroller 1.0
Anniversary Anim Pack
  • Includes three of Hitman's previous animation/bugfix mods, Subtle Camera Motion, Iron Sights Recoil Animations, and Fire Animation Variants
  • Also includes some new previously unreleased animations
  • Download and install the Main File - Anniversary Anim Pack 1.9c through MO2
  • Once the mod has been installed, double-click it in the left pane of MO2 and select Filetree
  • Delete every folder EXCEPT the characters folder
Anniversary Anim Pack - General Bugfix
  • Includes three of Hitman's previous animation/bugfix mods, Subtle Camera Motion, Iron Sights Recoil Animations, and Fire Animation Variants
  • Also includes some new previously unreleased animations
  • Main File - AnniAnimPack_BugFix 1.3
  • Main File - Bonus Patch 1.3a
FNV Clean Animations
  • Includes three of Hitman's previous animation/bugfix mods, Subtle Camera Motion, Iron Sights Recoil Animations, and Fire Animation Variants
  • Also includes some new previously unreleased animations
  • Main File - FNV Clean Animations 2.2
  • Update File - FNV Clean Animations - Update 2.2.2
Visuals
Animated Player Interactions
  • Allows you to customize the light radius and RBG values of the Pip-Boy light
  • Settings in the INI file config/Configurable Pip-Boy Light Color.ini
  • Includes the functionality of SmoothLight - Pip-Boy Light Enhancer
Main File - Animated Player Interactions 1.0
Armed to the Teeth Redux Allows you to holster up to 4 weapons of any type in any of four locations on your body. No compatibility patches required. No biped slots used. Adds visible holsters to weapons. Main File - Armed to the Teeth - Redux 2.3
Configurable Pip-Boy Light Colour
  • Allows you to customize the light radius and RBG values of the Pip-Boy light
  • Settings in the INI file config/Configurable Pip-Boy Light Color.ini
  • Includes the functionality of SmoothLight - Pip-Boy Light Enhancer
Main File - Configurable Pip-Boy Light Color 2.0
Improved LOD Noise Texture
  • Vastly improves the LOD noise texture used on all distant land
  • Uses only minimal video memory and greatly improves quality
 Main File - Improved LOD noise Texture V1.00
No More Motion Sickness Removes any kind of blur (motion blur, get hit blur, etc...) from the game. Main File - No More Motion Sickness (Script Runner) 1.2
PAVE TTW
Power Armor Visual Enhancement
  • Custom textures for all the power armors in FNV and F3/TTW
  • Also includes recon armor
  • Main File - Power Armor Visual Enhancement 2K 1.2
  • Update File - Power Armor Visual Enhancement Mesh Patch 1.0
  • Guide File - PAVE TTW 3.3 Patch
Securitrons on Alert Securitrons' faces will change when they enter combat mode. Main File - Securitrons On Alert 4.4.1
Lighting & Weather
A Little More Lamplight An enhancement to the shoddy work on the vanilla functional streetlamps & lights of Outer Vegas, Camp McCarran, Camp Golf, and NCRCF. Main File - A Little More Lamplight 1.2
Improved Lighting Shaders xNVSE plugin that almost completely fixes the exterior lighting bug, and allows up to 4x the number of active lights!
  • Optional File - Millenia's Environment Map Shader Fix Compatible Shaders 1.5beta4
  • Guide File - Improved Lighting Shaders - INI
SILO
Simple Interior Lighting Overhaul
  • Scripted simple solution for more immersive interiors. No FPS loss, no bloat
  • Automatic support for all interiors, including ones added by mods - including TTW
Main File - Simple Interior Lighting Overhaul 1.2
Desert Natural Weathers An overhaul of the weathers in New Vegas with 360 panoramic cloud textures, improved lighting, and DLC support. Main File - Desert Natural Weathers 3.3.5
Simple Natural Weathers TTW New natural looking weathers for all of the Capital Wasteland, Anchorage, Point Lookout, and The Pitt.
  • First install TTW Sun Fix
  • Main File - Simple Natural Weathers 0.99.1
